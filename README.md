Security Research Intern at **Huawei Research / Huawei Data Privacy Lab** (Düsseldorf), working on privacy-preserving authorization, anonymous authentication, and post-quantum cryptographic systems. MSc in Cybersecurity Engineering at **Politecnico di Torino** (GPA 29.48/30), expected October 2026.

My current research sits at the intersection of privacy-preserving technologies and post-quantum cryptography: secure credentials, anonymous authentication, zero-knowledge proofs, policy-aware access control, and long-lived security guarantees for AI-era systems.

I am especially interested in **agentic AI security** and in the broader agentic AI era: how agents use tools and APIs, how they handle context and permissions, how they can support cybersecurity workflows, and how agentic systems can be made safer, more auditable, more privacy-preserving, and more controllable.

My work also spans systems security and AI for cybersecurity, including Linux eBPF verifier research, malware and network-flow classification, SSH honeypot analysis, secure full-stack applications, and structured agent workflows.
